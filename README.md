# chaosrunner

DevOps tool for Kubernetes/Cloud infrastructure management.

## Quick Start

```bash
# Build
go build -o bin/chaosrunner ./cmd

# Run
./bin/chaosrunner --help

# Docker
docker-compose up --build
```

## Commands

- `chaosrunner` - Main command
- `chaosrunner version` - Print version

## Configuration

See `.env.example` for environment variables.
